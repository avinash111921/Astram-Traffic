# Astram: Smart Urban Traffic Incident Dashboard

This prototype is a data-driven system designed to analyze urban traffic incidents, identify congestion hotspots, and optimize traffic management resources. 

## Features
* **Geospatial Mapping:** Visualize active and resolved incidents across the city.
* **Predictive Insights:** Analyze peak hours for traffic disruptions.
* **Resource Allocation:** Track incidents by zone to deploy police/towing effectively.
